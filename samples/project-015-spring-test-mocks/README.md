# Mocks on spring project

Project sampling test scenarios using spring capabilities.

## Requirements

- java 21

## Project setup

Using [spring initializer][initializr].

## How to build

```bash
./mvnw package
```

## How to run

```bash
./mvnw springboot:run
```
